# Homework14

A simple SwiftUI app that shows an image of your favorite place.

## Features
- Double-tap the image to zoom in
- Long-press the image to zoom out
- Smooth zoom animations

## Setup
1. Create a SwiftUI project in Xcode
2. Add an image named `favoritePlace` to your Assets
3. Replace `ContentView.swift` with the provided code
4. Run the app
